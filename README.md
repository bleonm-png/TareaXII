# Sistema de Ventas en C++

Este proyecto consiste en un sistema básico de ventas desarrollado en C++ utilizando estructuras (`struct`), funciones y archivos binarios para almacenar información de productos.

## Funcionalidades

* Registro de productos
* Listado de productos
* Búsqueda por código y nombre
* Modificación de precio
* Actualización de stock
* Eliminación lógica de productos
* Proceso de venta con cálculo de IVA y actualización de stock

## Tecnologías utilizadas

* C++
* Archivos binarios (`.dat`)
* `ifstream`, `ofstream` y `fstream`

## Archivo utilizado

* `productos.dat` → almacena los productos registrados.
